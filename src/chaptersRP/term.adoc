| Контракт | Государственный контракт от 20 ноября 2025 г. № ГК 6401/25-4835 на выполнение работ по созданию государственной информационной системы: информационная система «Управление решениями экосистемы МосТех»

| Программное обеспечение | Совокупность приложений и программных документов, необходимых для эксплуатации этих программ
| Решение | Программное или аппаратное решение (или их сочетание), включая модули, библиотеки и компоненты, предназначенное для решения конкретных функциональных или бизнес-задач и обладающее ценностью для конечного пользователя в рамках информационной экосистемы города Москвы
| Профиль пользователя | Раздел Системы, который доступен пользователю после авторизации и содержащий информацию о пользователе, а также раздел настроек подписок и получения уведомлений
| Релиз (программный релиз) | Выпуск ПО, содержащий изменения и обновления
| Решения экосистемы | Серверное и пользовательское импортозамещенное ПО для сотрудников Правительства Москвы и сервисы для работы с ним
| Федеративное приложение | Приложение, состоящие из федеративных модулей
| Федеративный модуль/ модуль федеративного приложения | Расширение Webpack, представляющее собой комбинацию двух расширений: контейнера и ссылки на контейнер, позволяющие загружать части приложения из разных сборок во время выполнения
| Цифровая платформа (Платформа) | Комплексное программное решение, которое создает единую цифровую среду для взаимодействия множества независимых участников (пользователей, ИТ – специалистов и сотрудников Правительства Москвы)
| Экосистема МосТех | Совокупность взаимосвязанных сервисов для работы сотрудников, импортозамещенного аппаратного ПО и данных, цифровой платформы для разработки информационных систем и инфраструктурной основы, которые создают единую и бесшовную среду для работы пользователей, разработчиков и сотрудников Правительства Москвы
| API Gateway | Шлюз, который действует как единый интерфейс для запросов ко всем API системы: получает запросы API, применяет политики регулирования и безопасности, передает запросы в бэкенд-сервис, а затем передает ответ обратно запрашивающей стороне. Шлюз часто включает в себя механизм преобразования для организации и изменения запросов и ответов на лету. Шлюз также может предоставлять такие функции, как сбор аналитических данных и предоставление кэширования. Шлюз может предоставлять функциональность для поддержки аутентификации, авторизации, безопасности, аудита и соответствия нормативным требованиям
| JavaScript | Мультипарадигменный язык программирования. Поддерживает объектно-ориентированный, императивный и функциональный стили. Является реализацией спецификации ECMAScript (стандарт ECMA-262)
| Webpack | Сборщик модулей JavaScript с открытым исходным кодом
