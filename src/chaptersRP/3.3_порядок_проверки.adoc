Запустить веб-браузер.

В адресной строке указать адрес Системы. Открывается главная страница Системы для неавторизованного пользователя <<fig-pic1,рисунок 1>>.

В меню Системы нажать кнопку «Войти», после чего открывается страница аутентификации пользователя системы в СУДИР.

В форме аутентификации СУДИР ввести логин и пароль пользователя, нажать кнопку «Войти». Если данные введены корректно, будет выполнена аутентификация через СУДИР и отобразится главная страница.

Примечание – При вводе пароля действительное значение пароля заменяется символами «•». Для просмотра вводимого пароля необходимо отметить чекбокс «Показать пароль».

Если логин был указан некорректно, будет отображаться сообщение о некорректном вводе. Необходимо повторить ввод логина и пароля, указав верные данные и проверив нажатие клавиши CapsLock и раскладку клавиатуры.

Если пароль указан неверно, то после нажатия кнопки «Войти» будет выдано сообщение о неверном пароле. Необходимо указать корректный пароль и повторно нажать кнопку «Войти».
